# API сервис для работы с балансом кошельков

## Описание проекта:  

### Краткое описание:  
 
Проект позволяет делать запросы к эндпоиту api/v1/wallets/{WALLET_UUID}/operation для изменения баланса кошелька и к эндпоинту api/v1/wallets/{WALLET_UUID} для получения информации о кошельке. Эндпоинты покрыты тестами, запуск проекта осуществляется через docker-compose.yml. При запуске прект подключается к к серверу для разрабоки http://0.0.0.0:8000. Проект составлен с учетом работы в конкурентной среде, для запуска используются volumes.

---

### Использованные технологии:  


- Python 3.7.3
- Django 3.2.25 
- DjangoRestFramework 3.15.1
- PostgreSQL  


Запуск проекта осуществляется с помощью docker.  

---

## Запуск роекта:  


Клонировать репозиторий на свой компьютер:  

```bash  
git https://github.com/balahoncevg/wallets_api_server.git 
```

Выполнить команду в дирректрии проекта:   

```bash  
sudo docker-compose up --build -d 
```

**Примечание:**  
При создании контейнера автоматически применяются миграции, запускаются тесты и происходит подключение к серверу для разрабоки http://0.0.0.0:8000.

---

## Использование:  

Проект позволяет совершать следующие запросы:

POST запрос к эдпоинту

http://0.0.0.0:8000/api/v1/wallets/{WALLET_UUID}

- позволяет создать новый кошелек. Принимает данные вида {"balance": 10000} - где "balance" - это первоначальный баланс кошелька. Возвращает данные вида {"id": "WALLET_UUID", "balance": "10000.00"}, где "WALLET_UUID" - это uuid4 индентификатор кошелька.


GET запрос к эдпоинту

http://0.0.0.0:8000/api/v1/wallets/{WALLET_UUID}

- позволяет получить текущий баланс кошелька. Возвращает данные вида {"id": "WALLET_UUID", "balance": "10000.00"}.


POST запрос к эдпоинту

http://0.0.0.0:8000/api/v1/wallets/{WALLET_UUID}/operation

- позволяет изменить баланс кошелька. Принимает данные вида {"operationType": "DEPOSIT", "amount": 1000}, где "operationType" - это тип операции (пополнение "DEPOSIT" или списание "WITHDRAW") и "amount" - сумма, на которую изменится баланс. возвращает данные вида {"id": "WALLET_UUID", "balance": "11000.00"}.


**Примечание:**
Обратите внимание, что запросы принимаются без символа "/" на конце. Запросы не требуют авторизации и доступны любому пользователю.
